# brainrub
Simple brainfuck interpreter in Ruby

Usage: **./brainRub.rb programName.bf**

Example programs:

**cat**
```
,[.,]
```

**Hello World!**

```
>++++++++[<+++++++++>-]<.>>+>+>++>[-]+<[>[->+<<++++>]<<]>.+++++++..+++.>>+++++++.<<<[[-]<[-]>]<+++++++++++++++.>>.+++.------.--------.>>+.>++++.
```
